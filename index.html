About Me

I am a PhD Candidate in the Analytics of Software, GAmes, and Repository Data (ASGAARD) Lab at the University of Alberta. My research is focused on visual testing for web applications that are built with the HTML5 <canvas>, and I am supervised by Dr. Cor-Paul Bezemer.

Searching Software Engineering Papers

This project is a forked version of karpathy/arxiv-sanity-lite, adjusted to instead crawl and serve links to software engineering papers. This project also leverages ZeinabAk/DatasetSEVenues to include links to papers from software engineering journals and conferences. The website is intended to complement Google Scholar and arXiv.
